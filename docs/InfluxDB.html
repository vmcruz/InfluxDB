InfluxDB
Descripción
Permite crear una nueva instancia de InfluxDB
Propiedades
hostString Servidor al que InfluxDB debe conectarse (dirección ip o nombre de dominio)
portInteger Puerto al que se ha de conectar
userString Usuario que se usa para la conexión
passwordString Contraseña de acceso
dbString Base de datos a la que InfluxDB se conecta
errnoInteger Indica el código de error generado, en caso de que lo hubiera
errstrString Indica el mensaje de error generado, en caso de que lo hubiera
errtypeString Indica el tipo de error generado (librería, curl o servidor), en caso de que lo hubiera
precisionString Almacena la precisión con la que InfluxDB representará el tiempo (timestamp)
Parámetros
dbString Base de datos a la que se desea conectar de InfluxDB
userString Usuario de la base de datos
passwordString Contraseña del usuario de la base de datos
hostString Servidor donde se encuentra InfluxDB. Opcional, por defecto "localhost"
portString Puerto de la API HTTP de InfluxDB. Opcional, por defecto "8086"
Retorna
  Atención
  Advertencia
Ejemplos
Ej. Nueva instancia de InfluxDB: 
$influx = new InfluxDB("umidb", "umidev", "desarrollo123", "192.168.1.99");

Código fuente 
private $host;
 private $port;
 private $user;
 private $password;
 private $db;
 private $errno;
 private $errstr;
 private $errtype;
 private $precision;
  
 public function __construct($db = false, $user = "", $password = "", $host = "localhost", $port = "8086")  {
     if($db) {
         $this->host = $host;
         $this->port = $port;
         $this->user = $user;
         $this->password = $password;
         $this->db = $db;
         $this->precision = false;
     } else {
         $this->errno = 0;
         $this->errstr = "No database selected";
         $this->errtype = "lib";
     }
 }
